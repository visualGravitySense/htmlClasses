2. CSS-переменные для последовательного оформления

:root {
    /* Основные цвета */
    --primary-color: #3b82f6;
    --secondary-color: #10b981;
    --accent-color: #f59e0b;
    --danger-color: #ef4444;
    --dark-color: #1f2937;
    --light-color: #f3f4f6;
    
    /* Размеры и стили */
    --base-font-size: 16px;
    --heading-font-size: 24px;
    --border-radius: 8px;
    --box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1);
}

Темы на основе CSS-переменных

/* Светлая тема (по умолчанию) */
:root {
    --bg-color: #ffffff;
    --text-color: #333333;
    --highlight-color: #3b82f6;
}

/* Темная тема */
.dark-theme {
    --bg-color: #1f2937;
    --text-color: #f3f4f6;
    --highlight-color: #60a5fa;
}

/* Использование переменных */
body {
    background-color: var(--bg-color);
    color: var(--text-color);
}

.highlight {
    color: var(--highlight-color);
}
Примечание: CSS-переменные позволяют централизованно управлять цветовой схемой сайта, что облегчает поддержку и тематизацию.

Преимущества различных цветовых форматов

  • HEX (#RRGGBB): Компактный, широко используемый, удобный для копирования.
  • RGB/RGBA: Интуитивно понятный, легко добавлять прозрачность (альфа-канал).
  • HSL/HSLA: Наиболее удобный для человеческого восприятия, легко корректировать оттенок, насыщенность и яркость.

Когда что использовать:

  • Используйте HEX для твердых цветов без прозрачности, особенно когда работаете с дизайнерами.
  • Используйте RGB/RGBA когда нужна прозрачность или динамическое изменение каналов.
  • Используйте HSL/HSLA для создания цветовых схем и вариаций (осветление, затемнение, изменение насыщенности).
  • Используйте CSS-переменные для создания единой цветовой системы и тем.
Совет: HSL особенно удобен для создания оттенков цвета, так как можно просто менять последнее значение (яркость):
hsl(217, 91%, 30%)
hsl(217, 91%, 50%)
hsl(217, 91%, 70%)
hsl(217, 91%, 90%)
