Понимание блочной модели

/* CSS Блочной Модели */
.box-model-example {
    /* Контент */
    width: 200px;
    height: 100px;
    background-color: #3b82f6;
    
    /* Внутренний отступ (padding) */
    padding: 20px;
    
    /* Граница */
    border: 10px solid #1e40af;
    
    /* Внешний отступ (margin) */
    margin: 20px;
    
    /* Критически важное свойство */
    box-sizing: border-box;
}

/* Сравнение box-sizing */
.content-box {
    box-sizing: content-box;
}

.border-box {
    box-sizing: border-box;
}
/* HTML Структура */
<div class="container">
    <div class="box-model-example">
        Контент блока
    </div>
    <div class="another-element">
        Следующий элемент
    </div>
</div>
Пример блока с box-sizing: border-box
Общая ширина = 200px (контент + padding + border)
Блок с большими отступами
Демонстрация margin
content-box
Размер = контент + padding + border
border-box
Размер = фиксированный

Ключевые принципы блочной модели:

  • Контент: Основное содержимое элемента (текст, изображения, другие элементы)
  • Внутренний отступ (Padding): Очищенное пространство внутри элемента между контентом и границей
  • Граница (Border): Линия, окружающая padding и контент
  • Внешний отступ (Margin): Прозрачная область вокруг элемента, отделяющая его от других элементов
  • box-sizing: border-box позволяет включать padding и border в общую ширину/высоту элемента

Руководство по Типографике

Типографика - это комплексное искусство оформления текста, включающее выбор шрифтов, их размер, межстрочный интервал и другие параметры.

Основные принципы

  • Выбор подходящего шрифта
  • Правильные размеры и интервалы
  • Обеспечение максимальной читаемости

Правильная типографика улучшает визуальное восприятие контента.

Размеры шрифтов

xs - Очень маленький

sm - Маленький

base - Стандартный

lg - Большой

xl - Очень большой

Начертания шрифта

Thin - Очень тонкий

Light - Легкий

Normal - Обычный

Semibold - Полужирный

Bold - Жирный

Ключевые аспекты типографики:

  • Читаемость: Выбор шрифта и его параметров для легкого восприятия текста
  • Иерархия: Использование различных размеров и начертаний для структурирования контента
  • Адаптивность: Применение относительных единиц (em, rem) для гибкой типографики
  • Интервалы: Влияние межстрочного интервала (line-height) на удобочитаемость
  • Цветовой контраст: Правильное использование цвета для улучшения восприятия текста

Преимущества различных цветовых форматов

  • HEX (#RRGGBB): Компактный, широко используемый, удобный для копирования.
  • RGB/RGBA: Интуитивно понятный, легко добавлять прозрачность (альфа-канал).
  • HSL/HSLA: Наиболее удобный для человеческого восприятия, легко корректировать оттенок, насыщенность и яркость.

Когда что использовать:

  • Используйте HEX для твердых цветов без прозрачности, особенно когда работаете с дизайнерами.
  • Используйте RGB/RGBA когда нужна прозрачность или динамическое изменение каналов.
  • Используйте HSL/HSLA для создания цветовых схем и вариаций.
  • Используйте CSS-переменные для создания единой цветовой системы и тем.

1. Специфичность селекторов

Специфичность определяет, какое правило CSS применится к элементу, когда несколько правил конфликтуют. Более высокая специфичность имеет приоритет.

Тип селектора Пример Специфичность
Встроенные стили style="color: red;" 1,0,0,0

Классы, псевдоклассы, атрибуты .button, :hover, [type="text"] 0,0,1,0
Элементы и псевдоэлементы div, ::before 0,0,0,1

  • div.demo-box - lightgreen (0,1,1)

7. Принципы каскадирования

Порядок определения окончательного стиля элемента:

  1. Происхождение и вес: стили пользователя с !important > стили автора с !important > стили автора > стили пользователя > стили браузера
  2. Специфичность: если происхождение одинаково, выигрывает селектор с высшей специфичностью
  3. Порядок исходного кода: если специфичность одинакова, применяется последнее правило
