# UX UI / Frontend html

# HTML/CSS Curriculum for UX/UI Designers and Front-End Developers

## A 7-Lesson Comprehensive Course

---

## Lesson 1: Web Fundamentals & Modern Development Ecosystem

**For UX/UI Designers:**

- How the web works: Client-server architecture from a design perspective
- The design-to-development handoff process
- Understanding technical constraints that impact design decisions
- Design systems and their implementation in code
- Tools of the trade: Figma, Sketch, Adobe XD and their developer handoff features

**For Front-End Developers:**

- Modern development workflows and tooling
- Version control systems (Git) and collaboration practices
- Development environments setup (VS Code, extensions, linters)
- Browser developer tools for testing and debugging
- Package managers (npm/yarn) and build tools basics

**Practical Exercises:**

- Set up a development environment
- Create and share a simple design mockup
- Practice designer-developer communication with user story implementation

---

## Lesson 2: HTML5 Fundamentals & Semantic Structure



- How HTML structure influences user experience
- Semantic HTML and its impact on accessibility
- Content hierarchy and its relationship to HTML structure
- Design considerations for different HTML elements
- Understanding technical limitations of HTML elements



- HTML5 document structure and metadata
- Semantic HTML elements in depth (`header`, `nav`, `main`, `section`, `article`, `aside`, `footer`)
- Text content elements (`headings`, `paragraphs`, `lists`, `blockquotes`)
- Best practices for HTML organization
- Browser rendering process and performance implications

**Practical Exercise:**

- Convert a design mockup into semantic HTML structure
- Validate HTML for accessibility and semantics
- Peer review of HTML structure choices

---

## Lesson 3: CSS Fundamentals & Box Model

**For UX/UI Designers:**

- Translating design specifications to CSS properties
- Understanding the box model as it relates to layout design
- Typography in CSS vs design tools
- Color systems and implementation
- Identifying design patterns that translate well to CSS

**For Front-End Developers:**

- CSS selectors and specificity
- The box model in depth
- Units in CSS (px, em, rem, %, vh/vw)
- Typography and text styling
- Colors and backgrounds
- CSS organization and naming conventions (BEM methodology)
- Browser developer tools for CSS debugging

**Practical Exercise:**

- Style the HTML structure from Lesson 2
- Implement a consistent typography system
- Use CSS custom properties for design tokens

---

## Lesson 4: Responsive Design & Layouts

**For UX/UI Designers:**

- Responsive design principles from mobile to desktop
- Designing for different viewport sizes and devices
- Breakpoints and their strategic placement
- Progressive enhancement vs graceful degradation
- Wireframing responsive layouts

**For Front-End Developers:**

- Media queries and breakpoints
- Flexible layout techniques:
    - Flexbox for one-dimensional layouts
    - CSS Grid for two-dimensional layouts
    - Combining Flexbox and Grid
- Responsive images and media
- Mobile-first approach implementation
- Testing responsive designs across devices

**Practical Exercise:**

- Convert a fixed-width design to responsive layout
- Implement a complex grid layout with nested components
- Create a responsive navigation system

---

## Lesson 5: Advanced CSS & Modern Techniques

**For UX/UI Designers:**

- CSS capabilities for interactive elements
- Animation principles for web interfaces
- Microinteractions and their impact on UX
- Designing with CSS limitations in mind
- Dark mode and theming considerations

**For Front-End Developers:**

- CSS preprocessors (SASS/SCSS)
- CSS custom properties (variables)
- CSS transitions and animations
- Transforms and effects
- CSS functions (calc, min, max, clamp)
- Advanced selectors and pseudo-elements
- CSS methodology: ITCSS, SMACSS, or similar

**Practical Exercise:**

- Implement a theme switcher (light/dark mode)
- Create micro-animations for interactive elements
- Build an animated component without JavaScript

---

## Lesson 6: Forms & User Input

**For UX/UI Designers:**

- Form design best practices for usability
- Error states and validation feedback
- Accessible form design principles
- Progressive disclosure patterns
- Mobile-specific input considerations

**For Front-End Developers:**

- HTML5 form elements and attributes
- Custom form styling techniques
- Form validation (HTML5 and CSS)
- Accessible form implementation
- Mobile form considerations
- CSS-only custom form controls
- Form states (focus, valid, invalid, disabled)

**Practical Exercise:**

- Design and implement a multi-step form
- Style custom form elements consistently
- Create validation and error states using CSS

---

## Lesson 7: Performance, Accessibility & Best Practices

**For UX/UI Designers:**

- Accessibility in design (color contrast, focus states, etc.)
- Performance considerations in design decisions
- Design system documentation for developers
- Style guide creation and maintenance
- User testing techniques for UI implementation

**For Front-End Developers:**

- CSS performance optimization techniques
- Critical CSS and render-blocking resources
- CSS architecture for large projects
- WCAG accessibility guidelines implementation
- Semantic HTML and ARIA for accessibility
- Cross-browser compatibility strategies
- CSS feature detection and fallbacks

**Practical Exercise:**

- Audit and optimize a website for performance
- Perform accessibility testing and remediation
- Create a simple component library with documentation

---

## Course Project

Throughout the 7 lessons, students will work on a progressive project that builds with each lesson:

1. Create wireframes and mockups for a responsive website
2. Build semantic HTML structure
3. Implement core CSS styling
4. Make the site fully responsive
5. Add advanced CSS features and interactions
6. Implement and style forms
7. Optimize for performance and accessibility

Final deliverable: A fully responsive, accessible website with documentation of design decisions and technical implementation.
