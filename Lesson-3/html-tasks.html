HTML/CSS Tasks

These tasks are designed to help beginner practice HTML and CSS while focusing on UX/UI principles.

Beginner

Task 1: Create a User Profile Card

Learning Objectives:

  • Understand basic HTML structure
  • Learn CSS box model and layout
  • Practice visual hierarchy
  • Implement basic responsive design

Create a user profile card that includes:

  • Profile picture
  • User name
  • Short bio
  • Social media links

UX Focus: Ensure the card is visually balanced and the information hierarchy is clear.

Beginner

Task 2: Design a Navigation Menu

Learning Objectives:

  • Learn HTML lists and links
  • Understand CSS positioning
  • Practice hover states and transitions
  • Implement basic accessibility

Create a navigation menu that:

  • Is responsive (works on mobile and desktop)
  • Has clear visual feedback on hover
  • Includes proper spacing and alignment
  • Uses semantic HTML

UX Focus: Ensure the navigation is intuitive and easy to use.

Intermediate

Task 3: Build a Contact Form

Learning Objectives:

  • Learn HTML form elements
  • Understand form validation
  • Practice error handling
  • Implement user feedback

Create a contact form that includes:

  • Name, email, and message fields
  • Input validation
  • Clear error messages
  • Success/error states

UX Focus: Ensure the form is user-friendly and provides clear feedback.

Intermediate

Task 4: Design a Product Card

Learning Objectives:

  • Learn CSS Grid/Flexbox
  • Understand card design principles
  • Practice hover effects
  • Implement responsive images

Create a product card that includes:

  • Product image
  • Product name and price
  • Add to cart button
  • Hover effects

UX Focus: Ensure the card is visually appealing and encourages user interaction.

Advanced

Task 5: Create a Dashboard Layout

Learning Objectives:

  • Master CSS Grid layout
  • Understand responsive design
  • Practice component organization
  • Implement consistent spacing

Create a dashboard layout that includes:

  • Header with navigation
  • Sidebar menu
  • Main content area
  • Responsive grid system

UX Focus: Ensure the layout is intuitive and provides easy access to all features.

Advanced

Task 6: Design a Modal Dialog

Learning Objectives:

  • Learn CSS positioning
  • Understand overlay techniques

Create a modal dialog that includes:

  • Overlay background
  • Close button
  • Content area
  • Animation effects

UX Focus: Ensure the modal is non-intrusive and easy to dismiss.
